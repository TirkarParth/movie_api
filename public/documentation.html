Movie API Documentation

Project Title:
MYFLIX APPLICATION
Building the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Documentation

Project Brief A2

In 2.1 exercise: Installed node.js open the terminal and run:

xcode-select --install

to install additional command line tools for Xcode needed for installing nvm (Node Version Manager) and eventually NodeJS. You’ll be asked to enter your system user account password. Run the following command in terminal to install and update the script from the following link:

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

Now it's time to create a directory using the following code in terminal:

mkdir movie_api

Navigate into the "movie_api" directory:

cd movie_api

Create a JavaScript file called "test.js" within the "movie_api" directory:

touch test.js

In EX 2.2: Create the following files:

  • documentation.html
  • test.js
  • index.html
  • server.js
  • log.txt
